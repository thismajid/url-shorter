## URL Shorter

This app can your URLs shortener and you can manage your URLs shortener.

## Technologies

Node.js / NestJS<br />
MongoDB<br />
EJS Template Engine<br />
Bootstrap 5<br />
jQuery<br />

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
